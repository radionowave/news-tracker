# News Tracker

Скрипт для трекінгу новинних блоків на головній сторінці сайту та експорту метаданих у CSV файл. Використовує бібліотеки `requests`, `BeautifulSoup` та `Gradio` для створення візуального інтерфейсу.

   cd news-tracker
