# Інструкція до скрипта моніторингу веб-сайтів

Цей скрипт на Python призначений для моніторингу веб-сайтів на наявність певних текстових фрагментів. Він дозволяє відстежувати, коли текст з'являється або зникає зі сторінки. Ось основні функції та інструкції з використання:

1. Ініціалізація:
   - Переконайтеся, що у вас встановлені всі необхідні бібліотеки (os, sqlite3, time, csv, requests, beautifulsoup4, gradio).
   - При першому запуску скрипт створить базу даних SQLite з назвою `news_aggregator.db`.

2. Додавання моніторингу:
   - Перейдіть на вкладку "Додати/Оновити монітор".
   - Виберіть "Add new..." у випадному списку "Назва моніторингу".
   - Введіть URL сторінки, яку хочете моніторити.
   - Введіть текстовий фрагмент, який хочете відстежувати.
   - Введіть унікальний цифровий ідентифікатор статті.
   - Натисніть "Submit" для додавання.

3. Оновлення моніторингу:
   - На вкладці "Додати/Оновити монітор" виберіть назву існуючого моніторингу зі списку.
   - Змініть URL, текстовий фрагмент або ідентифікатор статті за потреби.
   - Натисніть "Submit" для оновлення.

4. Перегляд моніторів:
   - На вкладці "Монітори" ви побачите таблицю з усіма вашими моніторами.
   - Таблиця показує текстовий фрагмент, дату початку моніторингу, статус (активний чи ні) та ідентифікатор статті.
   - Червоний текст вказує, що фрагмент перестав відображатися, із зазначенням часу.

5. Експорт даних:
   - Перейдіть на вкладку "Експорт даних".
   - Натисніть "Submit", щоб експортувати всі дані моніторингу у CSV файл.
   - Файл `monitors_export.csv` буде створено в тій же папці, де й скрипт.

6. Імпорт даних:
   - На вкладці "Імпорт даних" виберіть CSV файл для завантаження.
   - Файл повинен мати ту ж структуру, що й експортований файл.
   - Натисніть "Submit" для імпорту даних.

7. Очищення бази даних:
   - На вкладці "Очистити базу даних" виберіть "Так" або "Ні".
   - Будьте обережні: вибір "Так" видалить усі дані з бази даних.

8. Видалення окремого монітора:
   - Перейдіть на вкладку "Видалити монітор".
   - Введіть цифровий ідентифікатор статті, яку хочете видалити.
   - Натисніть "Submit" для видалення.

Цей інструмент допоможе вам ефективно відстежувати зміни на веб-сайтах, особливо коли вам потрібно знати, коли певна інформація з'являється або зникає.
